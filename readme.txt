BW Picasso

Features

Application to convert all images (now including webp) to jpg and/or to webp

The support to webp files are provided by the Google (compiled by us) dlls
cwebp_ar.dll and dwebp_ar.dll (sources included, older versions of this
project was using the Google executables.

Requires:
Windows 7 or above and .net framework 4.5.1 installed

Ok12 says thanks to you for using our software.)

Important, the 64 bits SharpDevelop 5.1 project is
expecting the folder 'C:/AMANDA AVA KOCI PEREIRA/nosso_trabalho/amanda_projects/project picasso//'
available in your system, modify it as you wish, but the
project may not load if you don´t hack the xml project files
